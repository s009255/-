2.졸업 논문 미팅
-OpenCV
  --RGB 값을 hexa code 형태로도 저장을 해놓을 것.
    단순히 값을 hex로 바꾸려 하는 것은 numpy값에 기존 float형태의 값에서 string 형태로의 변환이라 불가능.
    numpy에 string을 저장할 수 있는 np.chararray를 만들어 값을 넣으려 했으나 chararray는 값을 b string 형태로 저장함.
    결국 빈 array에 hexa code로 변환한 값을 하나하나 넣는 것으로 해결.
  --색상 비교를 단적으로 두 개의 색상 비교를 하는 코드도 충분히 괜찮다. 두 이미지에서 clustering 결과로 나온 색상들은 각각 1:1로 비교하고
    그 결과 값에 두 값의 비중을 곱하여 더하는 것으로 유사도를 결정 지을 수 있을 것이다.
  --clustering의 속도가 느린 것도 해결해야 함.
    이는 이미지 사이즈를 줄이거나, blur처리 등을 활용하여 해결을 해나가야 할 듯.
